# EVAD Le Jeu

Serious game de découverte d'**EVAD** (Écosystème Vivant d'Action et de Développement).

En attendant la bêta, ce jeu fait découvrir EVAD en jouant : tout ce qu'on y manipule existe (ou existera) dans la vraie plateforme : les cartes de la Bibliothèque, les indicateurs ICI, les Graines, la Vadance et la Vadité, les trois profils, et Deva.

## Jouer

Ouvre simplement [`index.html`](index.html) dans un navigateur (double-clic ou via GitHub Pages). Aucune installation, aucun compte : un seul fichier autonome, la partie vit dans le navigateur.

## Le jeu en bref

Nous sommes en 2030, dans un futur solarpunk. Tu fais fleurir un territoire d'ici 2033, en incarnant l'un des trois profils du réseau **Territoires Vivants** :

- 🏡 **Pilote d'impact** : tu opères un lieu, personnalise-le, installe des solutions, mesure-les avec des ICI, tisse des boucles circulaires. Objectif : 30 points de Vadité.
- 🌿 **Bâtisseur d'impact** : tu contribues avec tes compétences, missions, apprentissages, Graines. Objectif : 5 missions.
- 🌾 **Semeur d'impact** : tu finances la régénération en euros et récoltes de la Vadité vérifiée. Objectif : 25 points de Vadité financée.

La **spirale VADE** (Valoriser · Activer · Développer · Essaimer) structure chaque partie ; les saisons et les aléas climatiques rythment le jeu ; **Deva**, l'assistante IA frugale, t'accompagne (avec l'empreinte de la conversation).

## Technique

Fichier HTML unique, sans build ni dépendance externe (hors police Satoshi et fond de carte). Direction artistique calquée sur le prototype EVAD.
